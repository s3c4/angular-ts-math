# angular-ts-math
An angular repository for numbers and math.

# Content

* [Install](#install)
* [Import](#import)
* [Use](#use)
* [License](#license)

# Install
```bash
npm install angular-ts-math
```

# Import
```typescript
import { angularMath } from 'angular-ts-math';
```

# Use
```typescript
angularMath.isEven(10);
```

```typescript
angularMath.isOdd(10);
```

```typescript
angularMath.sum(1, 2, 3);
```

```typescript
angularMath.sum(5, 6);
```
